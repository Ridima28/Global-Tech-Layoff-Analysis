# 📊 Tech Layoffs Analysis

This repository contains an exploratory data analysis (EDA) of global tech layoffs till 2025. The project analyzes layoff trends across companies, industries, countries, and time to uncover key patterns and insights.

## 📂 Repository Structure

```
Tech-Layoffs-Analysis/
│
├── Main/
│   ├── layoff-eda.ipynb               # Main Jupyter notebook
│   ├── README.md                      # Detailed EDA findings and insights
│   ├── tech_layoffs_till_2025.csv     # Dataset used for the analysis
│   └── Visuals/                       # Charts and visualizations generated during the analysis
│
└── README.md                          # Repository overview (this file)
```

## 📁 Folder Overview

### 📓 `main/`
Contains all project files related to the analysis.

- **layoff-eda.ipynb**
  - Complete data cleaning, preprocessing, exploratory data analysis (EDA), and visualizations.

- **README.md**
  - Detailed summary of the analysis, including key findings and answers to the business questions explored during the project.

- **tech_layoffs_till_2025.csv**
  - The dataset used for the analysis.

- **visuals/**
  - Collection of plots, charts, and other figures generated throughout the analysis.

## 📊 Project Highlights

The analysis explores questions such as:

- Which year experienced the highest number of layoffs?
- Which industries were most affected?
- Which companies laid off the most employees?
- How did layoffs change over time?
- Which countries were most impacted?
- Are there seasonal patterns in layoffs?
- How concentrated are layoffs across companies?

A detailed summary of these findings is available in `main/README.md`.
